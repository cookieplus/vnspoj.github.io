Cho số S = 111...11 (n chữ số 1, hệ thập phân), tính S^2.

#### Input

+ Dòng đầu tiên: số lượng test k.

+ k dòng tiếp, mỗi dòng ghi số n - số lượng chữ số 1 của S. (1 <= n <= 1000000)

#### Output

+ Với mỗi test ghi kết quả trên 1 dòng.

#### Example

```
Input
2
1
2

Output
1
121

```
